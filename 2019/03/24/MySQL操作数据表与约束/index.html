2. MySQL约束

2.1 空值与非空

NULL, 字段值可以为空
NOT NULL,字段值禁止为空

2.2 AUTO_INCREMENT

自动编号，且必须与主键组合使用
默认情况下，起始值为1，每次增量为1

2.3 主键约束(PRIMARY KEY)

每张数据表只能存在一个主键
主键保证记录的唯一性
主键自动为NOT NULL

2.4 唯一约束(UNIQUE KEY)

唯一约束可以保证记录的唯一性
唯一约束的字段可以为空值(NULL)
每张数据表可以存在多个唯一约束

2.5 默认值(DEFAULT)

当插入记录时，如果没有明确为字段赋值，则自动赋予默认值。
